# Contributing to POE2 How To Craft

Thank you for your interest in contributing! Your help is welcome, whether it’s reporting bugs, suggesting features, or submitting code improvements.

---

## How to Contribute

### 1. Fork the Repository
Click the **Fork** button on GitHub to create your own copy of the repository.

### 2. Clone Your Fork
Download your fork to your local system.

### 3. Create a New Branch
Always create a branch for your changes, e.g. `feature/your-feature-name`.

### 4. Make Changes
- Add features, fix bugs, or improve the GUI/probabilities.
- Follow existing code style.
- Add docstrings/comments where appropriate.

### 5. Test Your Changes
Ensure your changes do not break existing functionality and everything runs as expected.

### 6. Commit and Push
Commit your changes with a clear message, then push them to your branch.

### 7. Open a Pull Request
- Go to your fork on GitHub and open a pull request.
- Describe clearly what your changes do.
- Wait for review and discussion before merging.

---

## Bug Reports / Feature Requests

- Use the **Issues** tab on GitHub.
- Include:
  - Steps to reproduce the bug
  - Screenshots if relevant
  - System info (Java version, OS)

---

## Guidelines

- Keep commits clear and concise.
- Don’t push directly to the `main` branch.
- Be respectful in discussions.
